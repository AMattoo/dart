Vision

DART follows a Model-Driven Software Engineering approach with four stages: (i) Design the software using domain specific languages with precise syntax and semantics; we use a combination of two languages -- the Architecture Analysis & Design Language (AADL) and the DART Modeling and Programming Language (DMPL); (ii) Verify timing and functional properties of the software necessary for the system to work safely and effectively; (iii) Build the system by auto-generating code from AADL and DMPL to C++ targeted to a specific platform like a robot or a simulator; (iv) Run the software on the target platform. To implement this approach seamlessly, we are developing new ideas (more) and tools (more).
